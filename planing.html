GIT y GITHUB

Objetivo general

el objetivo de este trabajo es de diseñar un OVI y un OVA para el curso de diseños sitios web y conocer a configurar GIT y GITHUB

Objetivos especificos

  • Instalar git y añadir un repositorio y crear una confirmación.
  • Crear una cuenta GitHub
  • cononocer como utilizar las propiedades basicas de GIT y GITHUB

Materiales

GitHub es un sitio web y un servicio en la nube que ayuda a los desarrolladores a almacenar y administrar su código, al igual que llevar un registro y control de cualquier cambio sobre este código. basicamente lo que realiza es de publicar lo que se encuentra en nuestro servidor local y llevalo al servodor remoto GITHUB https://desarrolloweb.com/articulos/introduccion-git-github.html https://blog.muktek.com/git-y-github-627858e07167

introduccion a GIT y GITHUB

Como se ha dicho, Github github.com es un servicio para alojamiento De repositorios de software gestionados por el sistema de control de versiones Git. Por tanto, Git es algo más general que nos sirve para controlar el estado de un desarrollo a lo largo del tiempo mientras que Github es algo más particular un sitio web que usa Git para ofrecer a la comunidad de desarrolladores repositorios de software.
